# KoopCE 仓库论文报告

本文基于当前 `KoopCE` 仓库中的代码、notebook、已保存模型和结果图，对其最可信的研究主线进行了系统整理。综合各分支的代码完备度、实验工件数量以及方法与结果之间的连贯程度，当前最适合作为论文主体的方向，是“深度 Koopman 自编码器 + 奇异谱粗粒化 + 宏观动力学解释”。

与直接把仓库描述成一个泛化的动力学工具箱相比，这种更窄的 framing 更稳妥。原因在于：深度 Koopman 分支不仅有明确的模型实现和损失函数，也有保存下来的 toy 与空气质量 checkpoint、宏观变量可视化，以及后续的符号回归分析。相比之下，ResDMD、LoRA 与神经科学分阶段建模这些分支虽然同样有研究价值，但目前更适合作为补充材料或未来工作，而不是主论文的等权贡献。

## 一、主方法重建

仓库主方法的计算流程可以概括为：

1. 用编码器把微观观测 `x_t` 映射到潜变量 `z_t`。
2. 在潜空间中用同一个线性算子 `K` 重复推进，得到未来多个时刻的潜在状态。
3. 用解码器把潜状态恢复为原空间预测。
4. 在训练后对 `K` 做奇异值分解，用前几个左奇异向量构造宏观变量。

如果记编码器为 `g_theta`，解码器为 `h_theta`，则主干关系是

`z_t = g_theta(x_t),   z_hat_{t+tau} = K^tau z_t,   x_hat_t = h_theta(z_t)`。

仓库中最关键的不是“用神经网络拟合时间序列”这件事本身，而是训练后继续把 `K` 当作可分析对象。具体地，若

`K = U Sigma V^T`，

则宏观变量通过

`y_t = z_t U_{:,1:r}`

得到。也就是说，宏观状态不是额外训练出来的，而是从学到的潜在线性算子的主导奇异方向中提取出来的。

训练目标方面，代码中可以直接重建出如下总损失：

`L = alpha_1 (L_pred + L_rec) + L_lin + alpha_2 L_inf + alpha_3 ||K||_1 + alpha_CE L_CE`。

这里分别对应未来预测误差、重构误差、潜空间线性一致性、无穷范数鲁棒项、L1 正则，以及仓库中称为 CE 的谱整形项。当前代码明确实现了两种 CE 形式：

`L_CE^(1) = - sum_i sigma_i(K),`

`L_CE^(2) = - sum_i sigma_i(K) log sigma_i(K)`。

仓库里虽然还保存了一个带 `type3` 标签的 checkpoint，但当前 `CE_loss` 代码并没有 `type3` 分支，因此它只能被当作已有实验工件，而不能被纳入方法定义。

## 二、最有力的实验结论

### 1. PM2.5 单变量空气质量分支最适合作为主结果

长三角空气质量数据是当前仓库里证据最扎实的真实数据实验。该数据集包含 70128 个时间步、127 个站点，深度分支主要使用两种输入方式：

- 127 维 PM2.5 单变量输入
- 254 维 PM2.5+O3 联合输入

对 PM2.5 单变量分支中保存下来的 checkpoint 做统一复核后，可以得到一个非常有价值的结论：温和的 CE 谱整形确实能够改变潜在奇异谱的分布结构，而且这种变化并不一定会显著损害 rollout。

在 seed 匹配的两组 checkpoint 中：

- 无 CE 模型的 EC50 为 `4.9270`，5 步 forecast loss 为 `185.90`
- type1 CE 模型的 EC50 为 `5.3994`，5 步 forecast loss 为 `195.74`
- type2 CE 模型的 EC50 为 `5.3729`，5 步 forecast loss 为 `200.01`

这说明温和 CE 版本确实提高了谱贡献在前 50 个模态上的分布性，同时 rollout 代价只出现温和上升。

但这条结论并不是单向乐观的。另一些 CE-tagged checkpoint 表现出明显失稳：

- 一个 type1 变体的 forecast loss 达到 `4.30e8`
- 带 `type3` 标签的变体 forecast loss 达到 `1.73e4`

这些 checkpoint 同时伴随过大的头部奇异值。这表明仓库当前最有价值的负面发现同样非常清楚：谱整形不是越强越好，一旦主导模态被过度放大，潜空间线性推进会迅速放大误差。

### 2. Toy 分支更强地支持“可解释性”而不是“性能提升”

toy 系统使用了带显式二次耦合项的二维非线性映射，训练设置为 100 条训练轨迹、20 条测试轨迹和 30 步 rollout。对现有 toy checkpoint 的统一评估显示：

- 最好的已保存 rollout 误差约为 `1.081e-3`
- 其他 toy checkpoint 的 rollout 误差也基本维持在 `1e-3` 量级

也就是说，toy 分支中不同 checkpoint 的纯预测差异并不大。但这条分支最强的证据并不在数值误差，而在潜变量的符号化表达。仓库 notebook 中对编码器输出做 PySR 回归后，得到的表达式主要退化为关于第一个状态变量的简单二次组合，这与该 toy 动力系统中显式存在的二次耦合结构高度一致。

因此，toy 实验真正支撑的是“模型能够自动学到适合解释的提升观测函数”，而不是“某个 CE 配置显著优于其他配置”。

### 3. PM2.5+O3 联合分支说明多变量宏观建模是可行的

联合分支把 127 个站点的 PM2.5 与 O3 拼成 254 维输入。对两个已保存 checkpoint 的复核表明：

- 较旧联合模型的 5 步 forecast loss 为 `2116.91`
- 较强联合模型的 5 步 forecast loss 为 `317.14`

同时，较强模型的 EC50 也更高。这说明仓库的深度 Koopman 方案并不局限于单变量时间序列，而是已经具备处理耦合污染物的潜力。配合已有宏观变量图像，可以看出一维或二维宏观变量确实在跟踪多个站点之间共享的慢趋势。

不过这一分支目前还没有 CE 与无 CE 的成对对照，因此它更适合作为“多变量扩展方向已经初步跑通”的证据，而不是主论文的核心对比表。

### 4. 经典 SINDy / EDMD notebook 为主线提供了解释性支撑

仓库中的经典 notebook 虽然不属于同一端到端训练路径，但它们为“为什么要做粗粒化”提供了非常关键的支持。

- 提升后的非线性振子 notebook 报告 `mse = 3.281e-32`，几乎精确恢复目标方程。
- 双团 Kuramoto notebook 报告 `score = 0.9501695514710947`、`EC = 4.771670063172314`，并给出两个彼此独立的宏观线性递推方程。
- Kuramoto 网格扫参 notebook 共遍历 36 组参数，最高 EC 约为 `5.69`，并配有热力图、频谱与粗粒化系数分析图。

这些结果说明，只要观测函数选得足够好，宏观动力学就能变得低维、线性甚至非常易于解释。深度 Koopman 主线的意义，正是在于尝试自动学习这样一组观测函数。

## 三、建议的论文叙事

如果将当前仓库直接整理成论文，我建议标题与叙事都尽量收窄，聚焦在：

“学习具有谱结构的 Koopman 表征，并从潜在线性算子中提取粗粒化宏观动力学。”

围绕这一主线，论文正文最适合组织成以下结构：

1. 引言：强调从微观时间序列中发现宏观动力学的需求。
2. 方法：给出编码器-线性潜在推进-解码器结构，以及基于奇异向量的宏观变量构造。
3. 实验：
   - toy 系统：突出潜变量可解释性
   - PM2.5 单变量：突出温和 CE 的正面效果与激进 CE 的失稳边界
   - PM2.5+O3 联合：作为多变量扩展
   - 经典 sparse notebook：作为解释性验证
4. 讨论：强调当前仓库已经有足够强的 prototype 价值，但尚缺统一 run history 和 seed 平均统计。

在这个框架下，LoRA、ResDMD、神经科学阶段建模等内容可以放到讨论或附录里，作为“仓库中已经出现但尚未整合成统一实证矩阵的扩展方向”。

## 四、当前证据的边界

为了让这份报告保持可信，有几条边界需要明确写出来。

第一，仓库缺少统一的实验日志和 leaderboard。很多结论只能通过 checkpoint、notebook 输出和结果图反向恢复，而不是从一份系统化实验表直接读取。

第二，`type3` checkpoint 的真实训练目标在当前代码里无法复原，因此相关结果只能作为“现象”而不能作为“方法变体”来论证。

第三，部分 notebook 存在明显的乱序执行痕迹，意味着输出单元与当前可见代码顺序并不总是一一对应。

第四，toy 分支虽然很适合展示解释性，但不适合用来证明 CE 本身带来的稳定性能增益，因为保存下来的 checkpoint 混合了多种超参数变化。

## 五、结论

当前 `KoopCE` 仓库已经具备写出一篇“有研究味道、但结论需保持克制”的论文报告所需的核心材料。最稳妥的主线不是把所有分支都硬凑进一篇大而全的论文，而是聚焦深度 Koopman 自编码器与奇异向量粗粒化这一条证据最强的路线。

在这一主线上，仓库已经提供了三类重要信息：

- 方法上：完整的深度 Koopman 表征学习与粗粒化定义
- 实证上：PM2.5 单变量分支中温和 CE 的正面结果与激进 CE 的失稳边界
- 解释上：toy、Kuramoto 与提升振子 notebook 对宏观坐标和简洁方程的直接支持

如果后续能够补齐统一实验日志、明确各 checkpoint 的训练来源，并把空气质量分支做成 seed 平均的对照表，那么这套工作就很有机会从“研究原型”进一步推进到“论文闭环更完整的系统性工作”。
