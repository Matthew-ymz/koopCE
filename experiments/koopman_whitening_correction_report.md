# Koopman 白化修正报告

## 1. 修正目标

此前两个实验都把奇异谱分析建立在错误的“白化”对象上，核心问题是把

\[
G^{1/2} A G^{-1/2}
\]

误当成了标准的白化 Koopman 矩阵。  
对白化 Koopman 奇异值上界定理，真正对应的对象应为

\[
\bar K = C_{00}^{-1/2} C_{01} C_{11}^{-1/2},
\]

其中

- \(C_{00} = \mathbb E[\psi(x_t)^\top \psi(x_t)]\)
- \(C_{01} = \mathbb E[\psi(x_t)^\top \psi(x_{t+1})]\)
- \(C_{11} = \mathbb E[\psi(x_{t+1})^\top \psi(x_{t+1})]\)

并且离散一步算子应由配对样本直接拟合：

\[
\psi(x_{t+1}) \approx \psi(x_t) A,\qquad A = C_{00}^{\dagger} C_{01}.
\]

## 2. 代码修改

本次统一把两个实验改成了“配对样本拟合 + 双边白化”的流程。

### 2.1 公共 helper

在 `tools/tools.py` 中新增了：

- `inverse_sqrt_psd`
- `compute_transition_covariances`
- `whiten_operator_matrix`
- `fit_data_koopman_operator`

现在可以直接从一步配对样本得到：

- 数据驱动的一步离散算子 `A`
- 正确白化后的 Koopman 矩阵 `K_bar`
- `C00/C01/C11` 以及对应的平方根、逆平方根

### 2.2 Toy nonlinear example

Notebook: `experiments/toy_nonlinear/example_nonlinear_yb.ipynb`

修正点：

- 修复了 notebook 的导入路径，统一从仓库根目录导入模块
- 保留原来的连续时间 SINDy 拟合作为解析模型恢复
- 新增基于配对样本的离散一步 Koopman 拟合
- 奇异谱分析改为对 `K_bar = C00^{-1/2} C01 C11^{-1/2}` 做 SVD

### 2.3 Kuramoto two-group

Notebook: `experiments/kuramoto/kuramoto_2group_gram_v2.ipynb`

修正点：

- 原先的 `Gram -> G^{1/2} A G^{-1/2}` 被替换为配对样本统计 `C00/C01/C11`
- 保留连续时间 SINDy 模型，但其白化只作为“模型近似”对比
- 后续宏观变量提取改为使用 `C00^{-1/2} U_r`
- 宏观动力学方程改为使用数据驱动的一步算子 `A_step_data`

## 3. 修正后的结果

### 3.1 Toy nonlinear example

SINDy 在 lifted 空间里几乎精确恢复了解析方程：

- `mse = 3.281e-32`
- `model score = 0.9999999974`

关键奇异值结果如下：

| 对象 | 最大奇异值 |
| --- | ---: |
| 数据驱动一步算子 `K_data` | `1.008123` |
| SINDy 模型经正确白化后的矩阵 `K_bar_model` | `1.021495` |
| 经验白化 Koopman 矩阵 `K_bar` | `1.000000` |

更完整的谱：

- 原始一步算子奇异值：`[1.008123, 1.003095, 1.000000, 1.000000, 1.000000, 0.996915, 0.991943]`
- 白化后的奇异值：`[1.000000, 1.000000, 1.000000, 1.000000, 1.000000, 1.000000, 1.000000]`

解释：

- 这个 toy lifted 子空间本身就是封闭的、无损的，因此正确白化后奇异值全部塌到 1
- 之前看到略大于 1，来自“分析对象不是标准白化矩阵”，不是定理失效

### 3.2 Kuramoto two-group

配对样本协方差高度病态：

- `cond(C00) = 5.5659e+16`
- `cond(C11) = 7.4963e+16`

连续时间 SINDy 模型在 lifted 空间上的 one-step 近似与数据驱动离散一步算子偏差较大：

- `||A_step_data - A_step_model||_F = 9.055889`

关键奇异值结果如下：

| 对象 | 最大奇异值 |
| --- | ---: |
| 数据驱动一步算子 `A_step_data` | `5.613563` |
| SINDy 模型经正确白化后的矩阵 `K_bar_model` | `2.564635` |
| 经验白化 Koopman 矩阵 `K_bar` | `1.000000` |

前 10 个奇异值：

- 原始一步算子：`[5.613563, 4.457465, 1.617129, 1.592144, 1.560323, 1.548012, 1.444295, 1.276449, 1.230545, 1.205248]`
- 模型白化后：`[2.564635, 2.175449, 1.458179, 1.367513, 1.233053, 1.146702, 1.108278, 1.104041, 1.074467, 1.067820]`
- 经验白化后：前 10 个均为 `1.000000`

near-1 模态计数：

- `σ > 0.99` 的模态数：`32`
- `σ > 0.95` 的模态数：`32`

解释：

- 原先看到“白化后奇异值远大于 1”，本质上是因为分析的是错误对象
- 即便用同样的 `C00/C11` 去白化连续时间 SINDy 近似算子，奇异值仍可能超过 1，因为它不是由同一批配对样本直接诱导出来的 `C00^{\dagger} C01`
- 真正满足定理的，是经验白化矩阵 `K_bar = C00^{-1/2} C01 C11^{-1/2}`

## 4. 结论

这次修正后，两个实验都支持同一个结论：

1. 白化 Koopman 奇异值上界定理对应的是配对样本定义的经验矩阵 `K_bar`，不是任意拟合模型的相似变换版本。
2. 如果直接分析原始一步算子，或者分析与 `C00/C01/C11` 不一致的模型矩阵，即使做了“某种白化”，奇异值仍可能显著大于 1。
3. 对谱截断和宏观变量提取，更合理的对象应是经验白化 Koopman 矩阵；这时“保留 near-1 模态”的解释才与理论一致。
