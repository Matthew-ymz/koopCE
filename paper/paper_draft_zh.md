# KoopCE：面向粗粒化动力学的谱结构 Koopman 表征学习

## 摘要

本文基于当前仓库中的代码、模型权重、notebook 输出与已保存图像，重建其最可信的研究主线。现有证据最集中地支持这样一条方法路线：先通过深度自编码器学习非线性提升表征，再用线性潜在算子推进时序演化，最后通过奇异向量投影提取宏观变量。训练目标同时包含重构误差、多步预测误差、潜空间线性一致性、鲁棒项、稀疏正则项，以及仓库中记作 CE 的奇异谱正则项。当前最强的实验支持来自两个方向：二维合成非线性映射与 127 个站点的空气质量时间序列。在带有显式随机种子标记的 PM2.5 单变量模型中，温和的 CE 变体可将前 50 个奇异值上的谱熵型指标从 `4.9270` 提升到 `5.3994` 或 `5.3729`，同时 5 步 rollout 损失仅从 `185.90` 变化到 `195.74` 或 `200.01`；但更激进的 CE 配置会导致预测明显失稳，其 forecast loss 可上升到 `1.73e4` 乃至 `4.30e8`。在 toy 系统上，现有最优已保存 checkpoint 的 30 步 rollout 损失约为 `1.081e-3`，而对潜变量的符号回归给出了与目标系统一致的简单二次型观测函数。除此之外，Kuramoto 与提升后振子系统上的稀疏辨识 notebook 进一步表明，一旦观测函数选择得当，低维宏观动力学可以变得非常简洁甚至近乎精确。综合来看，该仓库已经提供了足够可信的证据去支持“谱整形有助于发现可解释宏观坐标”这一结论，但这种收益对具体 checkpoint 与超参数非常敏感，目前的证据强度仍明显高于“可解释性主张”，而弱于“全面稳定的性能提升主张”。

## 引言

### 问题背景

复杂非线性动力系统通常存在某种较低维的宏观描述，但如何仅从时间序列中自动发现这类描述始终是困难问题。一个真正有研究价值的方案不能只做短期预测，还应当满足至少三个条件：一是能学习到适合线性演化的潜在表示；二是能从潜在表示中提炼出具有宏观意义的变量；三是能进一步支持符号回归、稀疏辨识或谱分析等科学解释工具。当前仓库正是在这一交叉点上展开探索。

从整体结构看，仓库中并不只有一条研究线索，还同时包含深度 Koopman 自编码器、经典 SINDy/EDMD 管线、ResDMD 诊断、独立的 LoRA Koopman 实现，以及阶段化神经时间序列建模等探索性内容。为了形成一篇边界清楚、可被现有证据支持的论文报告，本文选择最窄但最稳妥的叙事：以深度 Koopman 自编码器与后续奇异向量粗粒化为主线，将其他分支视为支撑性分析或未来扩展。

### 贡献概述

基于当前仓库，本文能够较有把握地支持以下四点。

1. 仓库实现了一条完整的端到端管线：非线性编码、线性潜空间演化、解码重构，以及基于奇异向量的宏观变量提取。
2. 学到的 Koopman 线性算子并不只是预测工具，还被显式地用于奇异谱分析与粗粒化系数构造。
3. 在 PM2.5 单变量空气质量实验中，温和的 CE 型谱正则可以提高潜在奇异谱的分布性，而不显著破坏 rollout 质量；但更激进的谱整形会造成明显失稳。
4. 仓库中的经典稀疏辨识 notebook 强有力地支撑了“合适的观测函数可以让宏观动力学变得低维且可解释”这一核心思想。

## 方法

### 问题定义

记 `x_t in R^d` 为时刻 `t` 的微观观测状态，目标是在保留动力学信息的前提下学习一种更适合线性演化与宏观解释的表示。仓库中的主方法通过编码器 `g_theta`、线性潜在算子 `K` 与解码器 `h_theta` 完成这一任务，并在训练后借助 `K` 的奇异向量构造宏观变量。

### 记号系统

对于长度为 `S+1` 的观测窗口 `x_{t:t+S}`，定义

`z_t = g_theta(x_t),   z_hat_{t+tau} = K^tau z_t,   x_hat_t = h_theta(z_t)`。

其中 `k` 是潜空间维度，`r <= k` 是宏观变量维度，`K in R^{k x k}` 为可训练的线性潜在演化矩阵。

### 方法概览

主干模型由多层感知机编码器、线性 Koopman 潜在演化模块以及多层感知机解码器组成。与普通黑盒时序模型相比，这一路线的关键区别在于：仓库并不把潜在算子仅仅当作隐式参数，而是把它视为一个需要被分析的对象。训练完成后，其奇异值用于度量谱结构，其左奇异向量用于构造粗粒化投影，从而把微观状态压缩为可解释的宏观变量。

### 模块设计

编码器将原始观测映射到潜空间，解码器把潜变量恢复到原空间。核心创新不在网络结构本身，而在“潜空间线性推进 + 奇异谱后处理”的组合。对于每个样本，模型先计算初始潜变量 `z_t`，再重复应用同一个线性算子 `K` 得到未来多个时刻的潜在状态，最后再经解码器恢复预测序列。

训练结束后，对 `K` 做奇异值分解

`K = U Sigma V^T`，

并取前 `r` 个左奇异向量组成粗粒化矩阵

`C = U_{:, 1:r}`。

对任意潜变量 `z_t`，其对应的宏观状态定义为

`y_t = z_t C`。

这一步将“学到一个可预测的潜在线性系统”转化为“得到一个可分析、可解释的低维宏观系统”。

### 训练目标与流程

当前实现中的总损失可写为

`L = alpha_1 (L_pred + L_rec) + L_lin + alpha_2 L_inf + alpha_3 ||K||_1 + alpha_CE L_CE`。

其中 `L_pred` 对应解码空间上的未来预测误差，`L_rec` 对应重构误差，`L_lin` 约束编码后的未来潜变量与线性推进结果一致，`L_inf` 是基于无穷范数的鲁棒项，`||K||_1` 是线性算子的稀疏正则。当前代码中可直接观察到两种 CE 型谱正则：

`L_CE^(1) = - sum_i sigma_i(K),`

`L_CE^(2) = - sum_i sigma_i(K) log sigma_i(K)`。

也就是说，现阶段真正被实现的“CE”更像是一类奇异谱整形目标，而不是一个完全独立的外部指标定义。仓库中虽然还存在一个带 `type3` 标签的已保存 checkpoint，但当前代码并没有对应的 `type3` 损失实现，因此它只能被视为经验工件，而不能被纳入方法定义。

### 推理过程

推理时，模型先编码当前状态，再在潜空间中重复应用 `K`，最后解码得到未来序列。宏观分析则在训练后单独进行：选定截断阶数 `r`，利用 `U_{:,1:r}` 把潜变量投影到低维空间，随后可以对 `y_t` 做可视化、频谱分析、符号回归，或者进一步拟合宏观动力学方程。

仓库还在多个 notebook 中使用了一个谱熵型指标来刻画奇异谱的分布性。对奇异值 `sigma_1 >= sigma_2 >= ...`，代码先定义

`a_i = (1/i) sum_{j=1}^i log sigma_j,   Delta_i = a_i - a_{i+1}`，

再令

`EC = - sum_i p_i log_2 p_i,   p_i = Delta_i / sum_m Delta_m`。

因此，本文将其解释为“基于正向谱贡献增量的 Shannon 熵”，用来刻画不同尺度上的谱贡献是否更加分散。

## 实验

### 数据集与划分

本文主要使用三类证据。

第一类是二维合成非线性映射。仓库中的 toy notebook 使用 `lambda = 0.1`、`mu = 0.9`，生成 100 条训练轨迹、20 条测试轨迹，并以 31 步窗口执行训练和 30 步 rollout 评估。这一设置主要用于验证学到的潜变量是否具有可解释性。

第二类是长三角空气质量数据。仓库自带的数据集包含 70128 个时间步、127 个站点以及包括 PM2.5 与 O3 在内的多种气象/环境变量。深度 Koopman 分支分别考虑了 127 维 PM2.5 单变量输入与 254 维 PM2.5+O3 联合输入，并采用前 60000 个时间步训练、其余时间步测试的切分方式。

第三类是经典稀疏辨识 notebook，尤其包括提升后的非线性振子、双团 Kuramoto 模型以及 36 组参数的 Kuramoto 网格扫参。这些实验并非深度模型的严格对照组，但它们为“如何从观测函数走向可解释宏观动力学”提供了重要支撑。

### 对比方法

当前仓库并没有现成的外部基线总表，因此本文的对比主要来自仓库内部。

- PM2.5 单变量分支中的无 CE checkpoint 与 CE checkpoint 对比。
- PM2.5+O3 联合分支中两个成熟度不同的已保存 checkpoint 对比。
- toy 分支中若干带不同正则/隐藏层标签的 checkpoint 对比。
- 经典 SINDy/EDMD/ResDMD 管线作为分析性参照，而不是严格意义上的同一训练框架对照组。

这一点限制了结论的外推范围，但并不妨碍我们对谱整形、宏观提取与失稳现象作出有边界的判断。

### 评价指标

对于已保存 checkpoint 的复核，本文使用仓库现有损失函数与 rollout probe。空气质量实验中，`val_loss10` 表示前 10 个验证 batch 的平均损失，`forecast_loss_2048` 表示前 2048 个测试窗口上的 5 步 rollout 损失。toy 实验中，`forecast_loss` 是 30 步 rollout 指标。除此之外，还报告奇异谱上的 EC 指标与最大奇异值。对经典 notebook，则直接使用已保存输出中的 score、MSE 与宏观方程结果。

### 实现细节

PM2.5 单变量分支使用 `127 -> 512 -> 256` 的编码路径与 `256 x 256` 的潜在线性算子。PM2.5+O3 联合分支使用 `254 -> 512 -> 512` 的编码路径与 `512 x 512` 的潜在线性算子。toy 分支使用三维潜变量，隐藏层宽度为 8 或 4。空气质量 notebook 的主要时间窗参数为 `Sp = 30`，batch size 为 256，预测 probe 为 5 步；toy notebook 的主要评估窗同样使用 `Sp = 30`。

需要强调的是，这些结果来自现有仓库工件的统一复核，而不是来自一份完整、可追踪的实验日志。因此，下面的表格更适合作为“存档 checkpoint 的标准化探针结果”，而非完整超参数网格实验。

## 结果

### PM2.5 单变量空气质量模型

#### 实验目的

这一部分的核心问题是：CE 型谱整形能否改善潜在奇异谱结构，同时不过度损害深度 Koopman 模型的 rollout 质量。

#### 设置概述

我们对仓库中 PM2.5 单变量分支的已保存 checkpoint 做了统一复核。最干净的对比来自带显式 seed 标签的无 CE checkpoint 与 type1 CE checkpoint；type2 可作为另一个稳定参考，而其余 CE-tagged checkpoint 则主要用于观察失稳边界。

#### 主要发现

| 变体 | EC50 | 最大奇异值 | 验证探针损失 | 5 步 Forecast 损失 |
|---|---:|---:|---:|---:|
| PM25-noCE-seed | 4.9270 | 1.4706 | 796.55 | 185.90 |
| PM25-CE1-seed | 5.3994 | 1.3106 | 819.86 | 195.74 |
| PM25-CE2 | 5.3729 | 1.3046 | 821.44 | 200.01 |
| PM25-CE1-aggressive | 4.7485 | 2.5176 | 1.84e11 | 4.30e8 |
| PM25-CE3-aggressive | 4.8996 | 2.0498 | 5.89e4 | 1.73e4 |

稳定的 CE 变体相较于 seed 匹配的无 CE 模型，把 EC 从 `4.9270` 提升到了 `5.3994` 或 `5.3729`，而 5 步 rollout 损失仅有温和上升。但更激进的 CE 配置会把头部奇异值推得过大，并伴随显著的 rollout 崩溃。

#### 结果解读

这组结果最适合支持一种克制的结论：温和的谱整形确实能改变潜在谱的结构分布，而且这种改变在 EC 指标上是可见的；但谱整形并不是单调有益的，一旦头部模态被过度放大，潜空间线性推进就会把误差不断放大，最终导致 rollout 严重失稳。

#### 局限或注意事项

仓库没有为每个 checkpoint 提供完整的训练日志，因此这里比较的是“已保存工件”而不是严格可复现的完整超参数网格。尤其是带 `type3` 标签的 checkpoint，在当前代码中缺少对应损失函数定义，因此只能作为经验现象而不能作为方法论证的一部分。

#### 小结

当前仓库最强的实证结论不是“CE 一定提高预测性能”，而是“保守的谱整形可以在有限预测代价下改善宏观谱结构，而激进的谱整形会失稳”。

### Toy 非线性映射与符号化潜变量

#### 实验目的

toy 实验的核心价值不在于进一步压低预测误差，而在于检验学到的潜变量是否具有清晰的科学解释。

#### 设置概述

仓库在二维非线性映射上训练三维潜空间模型，然后对提取出的宏观坐标做符号回归。该系统本身包含显式的二次耦合项，因此适合检验模型是否能恢复出正确的提升观测。

#### 主要发现

| 变体 | EC | 最大奇异值 | 验证探针损失 | 30 步 Forecast 损失 |
|---|---:|---:|---:|---:|
| Toy-noCE | 0.9426 | 0.0399 | 0.011132 | 0.001823 |
| Toy-L1 | 0.9980 | 1.0178 | 0.008179 | 0.001081 |
| Toy-strong-spectrum | 0.7970 | 1.8281 | 0.011544 | 0.001869 |
| Toy-moderate-spectrum | 0.9069 | 0.3801 | 0.010377 | 0.001852 |

从预测角度看，带额外正则标签的 `Toy-L1` checkpoint 获得了最小的 30 步 rollout 损失。但更重要的是，notebook 中对潜变量的符号回归输出给出了主要依赖第一状态变量的简单二次表达式，这与该 toy 系统的设计结构高度一致。

#### 结果解读

这说明 toy 分支更强地支持“可解释表征学习”而不是“谱正则一定带来更优数值误差”。不同 checkpoint 之间的 rollout 差异并不大，但学到的潜变量确实可以塌缩为简单的二次观测函数，这正是 Koopman 粗粒化想要达到的效果。

#### 局限或注意事项

现有 toy checkpoint 混合了多种变化因素，包括隐藏层宽度、正则项以及不同标签的谱整形设置，因此它们不能干净地隔离出 CE 的因果效应。相比之下，符号回归结果反而是这一分支中最有说服力的证据。

#### 小结

toy 分支最有价值的结论是：学到的潜变量能够逼近简洁、结构正确的提升观测函数，从而为后续宏观动力学建模提供了可解释坐标。

### PM2.5+O3 联合建模与宏观变量提取

#### 实验目的

这一部分考察深度 Koopman 管线能否从单变量 PM2.5 扩展到 PM2.5 与 O3 的联合建模，同时仍然保留宏观变量的可解释性。

#### 设置概述

联合分支把 127 个站点的 PM2.5 与 O3 拼接成 254 维输入，并保存了两个无 CE checkpoint。我们对其进行了统一复核，并结合已有的宏观变量可视化逻辑与已保存图片进行分析。

#### 主要发现

较新的联合 checkpoint 将 5 步 rollout 损失从 `2116.91` 降低到 `317.14`，同时 EC50 从 `3.5801` 提升到 `4.3823`。已有宏观图像显示，一维或二维宏观变量能较好地跟随多个站点 PM2.5 时间序列中的共同慢趋势，而不是仅反映个别站点的高频波动。

#### 结果解读

这表明仓库中的宏观提取机制并不局限于单一污染物，而是有能力处理多污染物耦合场景。与此同时，两个联合 checkpoint 之间巨大的性能差距也说明：在高维输入下，时间窗设置与训练成熟度对稳定性影响更大。

#### 局限或注意事项

联合分支目前没有与之匹配的 CE 对照组，因此它能支持“多变量宏观建模可行”的结论，但还不足以支持“CE 在多变量场景中同样有效”的结论。

#### 小结

PM2.5+O3 联合分支已经提供了有价值的定性与定量线索，但从证据完整度看，它仍弱于 PM2.5 单变量分支。

### 经典稀疏宏观动力学分析

#### 实验目的

经典 notebook 的目标是验证：只要观测函数选得足够合适，宏观动力学就可以被压缩成简单且可解释的低维规律。

#### 设置概述

本文主要使用三类经典结果：提升后的非线性振子、双团 Kuramoto 模型以及 36 组参数的 Kuramoto 网格扫参。这些实验基于手工构造的观测函数库与稀疏回归，而不是端到端学习。

#### 主要发现

提升后的非线性振子 notebook 报告 `mse = 3.281e-32`，几乎精确恢复了目标动力学。双团 Kuramoto notebook 报告 `score = 0.9501695514710947`、`EC = 4.771670063172314`，并进一步给出二维宏观变量各自独立的对角线性递推关系。更大的 Kuramoto 网格扫参覆盖 36 组参数组合，并在打印结果中出现约 `5.69` 的高 EC 值，同时配有热力图、奇异谱比较与频谱统计图。

#### 结果解读

这些 notebook 从另一个角度支撑了深度主线的科学意义：一旦表示空间足够合适，宏观动力学确实可以变得低维、线性化且可解释。深度模型的价值，正是为这类后续分析自动提供候选宏观坐标。

#### 局限或注意事项

这类 notebook 使用的是手工设计观测函数与不同的数据生成流程，因此不能把它们直接当作深度 Koopman 模型的严格基线。它们更像是“分析后端”的有效性证明，而不是“端到端学习路线”的直接竞争者。

#### 小结

经典稀疏辨识分支最强地说明了这样一点：如果宏观坐标足够合适，低维方程的恢复可以非常准确，而这正是当前仓库深度分支试图自动实现的目标。

### 图像重点分析

空气质量宏观图像的重要性在于，它把“损失函数上的优化结果”转化成了“可直观看见的系统级慢变量”。这些图中，宏观变量并不是简单重复某一个站点的波动，而是在多条 PM2.5 站点曲线之上刻画出共享趋势，这为“宏观变量确实捕捉到系统共同结构”提供了直接视觉证据。

经典 SINDy 可视化图则承担另一种角色。它展示了仓库中也存在一条更加传统的动力学辨识路线，可以直接比较真实轨迹、预测轨迹与误差。这说明该仓库并不是单纯依赖深度黑盒预测，而是在尝试把学习式表示与显式方程辨识结合起来。

Kuramoto 网格扫参中的热力图和奇异谱对比图则提供了最接近“敏感性分析”的证据。虽然仓库没有把这些结果整理成统一 leaderboard，但从现有可视化中仍能看出 EC、粗粒化系数规模以及频谱统计会随耦合强度和噪声变化而系统性改变。

## 结论

就当前仓库状态而言，最稳妥、也最能被证据支持的论文叙事，是“学习具有谱结构的 Koopman 表征，并从学到的潜在线性算子中提取粗粒化宏观动力学”。这一主线在代码层面是清晰的，在实验层面也已经具备合成系统、空气质量数据与稀疏分析三个层次的支撑。

现有证据中，PM2.5 单变量分支给出了最重要的正反两方面结论。正面来看，温和的 CE 型谱整形能够提高潜在奇异谱的分布性，而不会显著恶化 rollout；反面来看，一旦谱整形过强，模型就会明显失稳。toy 与经典 notebook 则从可解释性角度强化了这条主线，说明学习到或设计好的宏观坐标确实有机会恢复成简洁的低维规律。

当前最需要补强的不是方法雏形，而是实验记录与证据组织。仓库还缺少统一的 run history、seed 平均表格、完整的 checkpoint 来源说明，以及更清晰的多分支定位。换言之，它已经是一套相当有研究味道的原型系统，但距离“结论充分、实验闭环完整”的正式 benchmark 论文还差一层系统化整理。
