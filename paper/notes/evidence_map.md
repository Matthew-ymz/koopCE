# Evidence Map

## Status Legend

- `observed`: directly supported by code, checkpoints, notebook outputs, figures, or raw data shipped in the repository
- `inferred`: faithful abstraction derived from the implementation or artifact structure, but not written as explicit prose in the repository
- `placeholder`: unsupported or still ambiguous
